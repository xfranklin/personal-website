Vimium — расширение для браузеров, на базе Chromium, существенно расширяющее возможности управления браузером клавиатурой. Команды для управления основаны на текстовом редакторе vim. Ссылки на расширение для [Chrome](https://chrome.google.com/webstore/detail/vimium/dbepggeogbaibhgnhhndojpepiihcmeb) и [Firefox](https://addons.mozilla.org/ru/firefox/addon/vimium-ff/">Firefox).

## Все команды
**Навигация на странице:**

<span class="key">j</span>, <span class="key">Ctrl</span> + <span class="key">e</span> - скроллинг вниз\
<span class="key">k</span>, <span class="key">Ctrl</span> + <span class="key">y</span> - скроллинг вверх\
<span class="key">gg</span> - скроллинг на самый вверх страницы\
<span class="key">G</span> - скроллинг в самый низ страницы\
<span class="key">d</span> - скроллинг вниз на пол страницы\
<span class="key">u</span> - скроллинг вверх на пол страницы\
<span class="key">h</span> - скроллинг влево\
<span class="key">l</span> - скроллинг вправо\
<span class="key">r</span> - перезагрузка текущей страницы\
<span class="key">yy</span> - скопировать текущий URL\
<span class="key">p</span> - открыть содержимое буфера, в текущей вкладке\
<span class="key">P</span> - открыть содержимое буфера, в новой вкладке\
<span class="key">i</span> - режим ввода\
<span class="key">v</span> - режим выделения\
<span class="key">c</span> - режим каретки в режиме выделения\
<span class="key">gi</span> - фокус на первой форме страницы\
<span class="key">f</span> - открыть список ссылок, и перейти\
<span class="key">F</span> - открыть список ссылок, и перейти на новую вкладку

**Навигация по вкладкам**

<span class="key">t</span> - создать новую вкладку\
<span class="key">x</span> - закрыть текущую вкладку\
<span class="key">X</span> - восстановить закрытую вкладку
<span class="key">J</span>, <span class="key">gT</span> - перейти на следующую вкладку cлева\
<span class="key">K</span>, <span class="key">gt</span> - перейти на следующую вкладку справа\
<span class="key">^</span> - перейти к последней вкладке\
<span class="key">g0</span> - перейти к первой вкладке\
<span class="key">g$</span> - перейти к последней вкладке\
<span class="key">yt</span> - создать дубликат текущей вкладки\
<span class="key">Alt</span> + <span class="key">p</span> - закрепить или открепить текущею вкладку\
<span class="key">Alt</span> + <span class="key">m</span> - включить или выключить звук на текущей странице\
<span class="key">H</span> - перейти назад (в истории)\
<span class="key">H</span> - перейти вперед (в истории)

**Другие команды:**

<span class="key">o</span> - перейти на определенный url, вкладку, или страницу из истории\
<span class="key">O</span> - перейти на определенный url, вкладку, или страницу из истории (в новой вкладке)\
<span class="key">b</span> - открыть закладку\
<span class="key">B</span> - открыть закладку в новой вкладке\
<span class="key">T</span> - поиск среди открытых вкладок\
<span class="key">/</span> - режим поиска на странице\
<span class="key">?</span> - подсказка
